# Project Roadmap: The Future of Structural Engineering

This roadmap tracks the evolution of the Auto-Pilot BOQ system from a specialized rebar tool to a full-featured structural takeoff platform.

## 🟢 Phase 1: Professional Foundations (In Progress)
- [x] Full-width workspace scaling
- [x] High-precision "Click-Move-Click" engine
- [x] Advanced Stirrup Zonation logic
- [x] Per-instance deletion/selection
- [ ] **Next**: PDF Export V2 (Visual reports)
- [ ] **Next**: LocalStorage persistence stability

## 🟡 Phase 2: Structural Depth (Short Term)
- [ ] **Slab Reinforcement V2**: Detailed grid vs. mesh layouts for oddly shaped slabs.
- [ ] **Roof Structure**: Implementation of truss templates and roofing area calculations.
- [ ] **Wall Modules**: Masonry, plastering, and wall reinforcement specs.
- [ ] **Advanced Snapping**: Angle snapping (45/90 degrees) and parallel beam alignment.

## 🟠 Phase 3: External Integration (Mid Term)
- [ ] **DXF/DWG Viewer**: Import CAD files as a background layer for tracing.
- [ ] **CSV Import/Export**: Sync templates with existing company price lists.
- [ ] **Material Library**: Pre-defined library of concrete grades, steel types (SD40, SD50), and labor costs.

## 🔴 Phase 4: Intelligence & Collaboration (Long Term)
- [ ] **Auto-Detect**: AI-assisted node detection from background CAD images.
- [ ] **Multi-Level Projects**: Support for multiple floors/levels in a single BOQ.
- [ ] **Cloud Sync**: Multi-user collaboration and project revision history.
- [ ] **Mobile Takeoff**: Native-feel touch controls for on-site verification.
